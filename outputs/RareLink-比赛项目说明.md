# RareLink 稀联：每个科室一台 DGX Spark 的联邦科研智能体

## 项目简介

RareLink 面向儿童高级别胶质瘤等小样本、多中心医学影像研究场景，解决“数据不能集中、站点分布不一致、研究流程难以复现、结果难以解释和审计”的问题。目标用户是医院科室科研团队、影像科研究者和多中心临床研究协调方。系统不提供诊断或治疗建议，而是把一项多中心研究组织成可审批、可执行、可审计的科研工作流：研究者提出问题，Agent Team 起草协议；各站点只产生受控汇总统计；DGX Spark 在本地运行 MONAI 三维分割与 NVIDIA FLARE 联邦训练；Agent 基于聚合指标解释平均表现、最弱站点表现和局限性；最终导出实验合同、任务日志、模型路径、指标和研究报告。

项目聚焦“儿童高级别胶质瘤多中心 MRI 分割研究”，而非泛泛覆盖所有罕见病。该方向同时具有小样本、四模态三维 MRI、站点异质性和隐私边界清晰等特点，能真实体现联邦科研基础设施的价值。比赛版本在一台真实 DGX Spark 上模拟三个逻辑科室，必须明确：这不是三家真实医院的临床验证，也不能据此宣称临床有效性。

## 核心创新与价值

RareLink 的创新不在于单独训练一个分割模型，而在于把“协议到证据”的科研闭环产品化。第一，系统把原始影像、标签和患者级字段限制在站点内；传给协调面和 Step 3.7 的只有经策略过滤的汇总统计、锁定合同和聚合指标。第二，选择策略时不只看平均 Dice，还把最弱站点 Dice、站点 Dice 标准差和 HD95 写入固定合同与 Agent 评审，使公平性不再只是报告结尾的文字。第三，Agent 不能绕过人工审批修改数据划分、主要终点或训练预算；协议、合同、实验、失败与重试均进入审计账本。第四，新增“评委证据驾驶舱”：用 90 秒路径呈现“数据留在科室—Spark 本地训练—FLARE 聚合—Agent 解释证据”，并可展示一次安全注入的训练失败如何变为 `FAILED_RETRYABLE` 后重新执行。

## Agent Team 协作

研究主任 Agent 将自然语言问题转成结构化研究协议；站点数据管家 Agent 仅生成最小分组阈值保护后的可行性统计；实验设计 Agent 提出 Local、FedAvg、FedProx 的公平比较合同；统计评审/证据叙事 Agent 从聚合指标中给出工程候选、最弱站点风险和局限性；隐私评审 Agent 检查患者字段与小样本字段是否被阻断；科研写作 Agent 汇编可导出的研究叙事。所有 Step 3.7 输入都经过字段策略检查。未配置在线模型时，系统使用确定性模板 Agent，保证演示不会因外部 API 不可用中断；在线 Step 3.7 已在完全合成的研究输入上完成 JSON 模式协议与四角色协作冒烟验证。

## 技术实现与 DGX Spark 适配

系统由 React 证据驾驶舱、FastAPI 控制面、SQLModel/SQLite 实验账本、NVIDIA FLARE 编排和 MONAI/PyTorch 三维分割任务组成。研究状态机限制从协议、可行性、合同、训练、结果复核到报告导出的转换；Spark 统一内存保护锁使同一节点的三个逻辑科室串行训练，避免多个三维任务同时抢占内存。真实节点验证环境为 NVIDIA DGX Spark GB10、ARM64、CUDA 13、PyTorch 2.10.0+cu130、MONAI 1.6.0 和 NVIDIA FLARE 2.7.2。项目已在该节点实际完成 CUDA 矩阵计算、MONAI 3D SegResNet 训练、三逻辑站点 FedAvg/FedProx 聚合、全局模型持久化，以及 API 后台任务到前端指标回写的闭环。

模型优化以稳定性和可复现性优先：使用 CUDA 自动选择、AMP、AdamW、固定种子、共享的小型 SegResNet 合同、缓存数据集、FedProx 异质性正则和单节点串行调度；训练任务会持续回写进度、日志、全局模型路径、运行时长和峰值 GPU 内存。实机单轮极小合成数据的数值只用于验证模型、损失、评估、聚合和 GPU 链路，不包装成医学性能。公开 MSD 脑肿瘤基准的数据下载、归档校验、SHA-256 溯源、标签映射和非 IID 分区已经实现；由于赛事节点到官方源吞吐过低，未把未完成下载伪造为实验结果。

## 限制与后续计划

当前版本是单 Spark 三逻辑站点的工程原型，尚未进行真实跨院通信、临床有效性验证、差分隐私会计、PACS/FHIR 接入或正式授权数据的多轮重复实验。下一阶段将以合规获取的 BraTS-PEDs 等儿童公开数据作为外部工程验证，并在每家真实医院部署独立的 Spark Client 与证书化 FLARE 通信。RareLink 的长期目标不是替代医生，而是成为“数据不出院、证据可回溯”的多中心科研操作系统。
